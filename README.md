# Gestión de Horas Semanales en Python

## Descripción

Este programa fue desarrollado en Python para calcular las horas trabajadas por un equipo durante una semana laboral. A partir de los datos registrados, el sistema calcula el total de horas por cada recurso y determina si su jornada corresponde a un horario estándar o a sobretiempo.

## Objetivo

Automatizar el cálculo de horas semanales de los colaboradores y clasificar su jornada laboral según el número total de horas trabajadas.

## Funcionalidades

- Registro de 4 recursos.
- Almacenamiento de horas trabajadas de lunes a viernes.
- Cálculo automático del total de horas semanales.
- Clasificación de la jornada laboral.
- Generación de reporte por cada recurso.

## Lógica de Negocio

- Si el total de horas es mayor a 40 horas, se clasifica como:
  - Sobretiempo
- Si el total de horas es igual o menor a 40 horas, se clasifica como:
  - Horario Estándar

## Tecnologías Utilizadas

- Python 3
